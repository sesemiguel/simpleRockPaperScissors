# Simple Javascript Rock-Paper-Scissors
A rock-paper-scissors game implemented in Javascript.
### Mechanics
(CHOICE1,CHOICE2) - WINNER<br />
      (1,3) - 1<br />
      (2,1) - 1<br />
      (3,2) - 1<br />
      (3,1) - 2<br />
			(1,2) - 2<br />
			(2,3) - 2<br />
      (1,1) - DRAW<br />
      (2,2) - DRAW<br />
			(3,3) - DRAW<br />
